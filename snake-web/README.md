# Neon Snake (Web)

一个纯前端、单文件可运行的贪吃蛇小游戏。

## 运行方式

1. 直接用浏览器打开同目录下的 `index.html`。
2. 无需安装依赖、无需构建工具、无需本地服务器。

## 操作说明

### 桌面端

- 方向键：控制蛇移动方向
- `Space` / `Enter` / `R`：开始或重新开始游戏（结束后可重开）

### 手机端

- 屏幕下方方向按钮：点击控制移动
- 游戏区域内滑动：按滑动方向控制移动
- 游戏结束后点击画面：重新开始

## 游戏规则

- 吃到食物得分 +1
- 撞墙或撞到自己则游戏结束
- 界面显示当前得分和历史最高分
- 最高分使用 `localStorage` 持久化保存
- 分数越高，移动速度越快（难度逐步提升）

## 视觉与交互特性

- 暗色霓虹风格（霓虹绿蛇 + 发光食物）
- 平滑动画过渡
- 游戏结束显示动态得分动画

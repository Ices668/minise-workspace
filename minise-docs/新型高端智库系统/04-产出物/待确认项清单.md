# 待确认项清单

## 1. 接口与数据源
- 论文库、专利库、标准库、舆情库、人才库的实际对接方式（API / 文件 / 数据库同步）
- 各数据源字段映射、更新频率、鉴权方式、失败重试策略
- 是否存在已有内部系统需要双向同步

## 2. 部署与环境
- 最终部署环境：内网、政务云、私有云或本地机房
- 是否要求国产化适配（操作系统、数据库、中间件）
- 网络分区、边界防火墙、堡垒机、审计要求是否已有统一规范

## 3. 性能与容量
- 并发用户规模
- 数据规模预估（论文、专利、标准、舆情、人才）
- 分析任务峰值、报告生成频率、预警消息量级

## 4. 安全与权限
- 角色划分是否还需细化（管理员、研究员、专家、审核人员、访客等）
- 是否有等保级别、保密要求、日志留存周期要求
- 敏感数据脱敏范围与导出审批规则

## 5. 交付口径
- 两份文档最终要偏“投标版 / 汇报版 / 实施版”中的哪一种
- 是否需要补图：系统架构图、部署拓扑图、数据流转图、业务流程图
- 是否需要把 Word 再压成更正式的公文语气版本

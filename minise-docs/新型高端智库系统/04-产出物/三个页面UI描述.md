# 媒体聚焦、技术趋势分析、技术动态预警 - 页面UI描述

> 用于交给Stitch绘制UI设计稿

---

## 页面一：媒体聚焦

### 1.1 页面整体布局

**页面结构：**
- 顶部导航栏（高度：64px）
- 左侧边栏（宽度：240px，可收起）
- 主内容区域（自适应）
- 右侧详情面板（宽度：400px，可展开/收起）

**配色方案：**
- 主色调：深蓝色 #1E3A5F
- 强调色：橙色 #FF6B35（用于预警、热点标记）
- 背景色：浅灰 #F5F7FA
- 卡片背景：白色 #FFFFFF
- 文字色：主文字 #333333，次要文字 #666666

---

### 1.2 顶部导航栏

**布局：从左到右**
1. **Logo区域**（左侧）
   - Logo图标 + 系统名称"新型高端智库系统"
   
2. **模块导航**（中间）
   - 标签项：研究需求分析 | 技术体系分析 | 技术趋势分析 | 领域技术线路 | 战略咨询报告 | 战略咨询成果 | 智能问数
   - 当前选中项："媒体聚焦"高亮显示
   
3. **用户区域**（右侧）
   - 搜索图标
   - 通知图标（带红点提示）
   - 用户头像 + 用户名

---

### 1.3 左侧边栏

**布局：上下结构**

**上部 - 媒体聚焦模块菜单：**
- 媒体聚焦总览（icon：dashboard）
- 时间序列分析（icon：timeline）
- 耦合网络（icon：network）
- 热点检测（icon：fire）
- 预警配置（icon：bell）
- 报告生成（icon：file-text）

**下部 - 快捷操作：**
- 新建监控任务按钮
- 最近访问记录

---

### 1.4 主内容区域 - 媒体聚焦总览面板

**布局：2:1 上下布局**

**上部 - 关键指标卡片（4个）：**
- 总监测媒体数：显示数字 + 趋势箭头
- 媒体关注热点数量：显示数字 + 同比增长百分比
- 情感偏向指标：三个环形图（正面60%/中性30%/负面10%）
- 媒体—技术耦合度总体评分：显示分数（0-100）+ 等级标签

**卡片样式：**
- 尺寸：宽度自适应，高度120px
- 圆角：12px
- 阴影：0 2px 8px rgba(0,0,0,0.08)
- 数字字号：32px，加粗
- 标签字号：14px

**交互：**
- 点击"媒体关注热点"卡片 → 展开热点列表（侧边栏形式）
- 点击"耦合度评分" → 弹出耦合分析摘要模态框

---

### 1.5 主内容区域 - 时间序列与趋势视图

**布局：全宽图表区**

**图表区域：**
- 图表类型：多折线图（支持多条曲线叠加显示）
- X轴：时间轴（可选择 日/周/月）
- Y轴：数值轴（自动适配数据范围）
- 图例：媒体提及数、情感分布、学术产出、投融资事件、政策关键词（不同颜色区分）
- 图表高度：300px

**筛选控件（图表上方）：**
- 时间范围选择器：下拉框 [最近7天/最近30天/最近90天/自定义]
- 数据源复选框：论文/专利/标准/舆情/投融资/政策
- 导出按钮：导出CSV/导出PNG

**交互：**
- 鼠标悬停显示详细数据点
- 支持框选放大
- 点击曲线图例可显示/隐藏该曲线

---

### 1.6 主内容区域 - 媒体—技术耦合网络视图

**布局：全宽网络图区域**

**网络图：**
- 节点类型1：媒体节点（圆形，蓝色，节点大小表示影响力）
- 节点类型2：技术主题节点（圆形，橙色，节点大小表示产出量）
- 连线：表示关联关系，线条粗细表示关联强度
- 线条颜色：渐变色（蓝→橙）表示关联度从低到高

**网络图工具栏（顶部）：**
- 布局切换：力导向 / 放射状 / 层级
- 筛选器：按地域 / 时间范围 / 情感过滤
- 操作按钮：局部放大 / 路径查找 / 导出

**网络图操作（底部）：**
- 节点点击 → 右侧弹出详情面板
- 节点拖拽 → 调整布局
- 鼠标滚轮 → 缩放

---

### 1.7 右侧详情面板

**触发条件：点击网络图中的节点时展开**

**面板内容结构：**
1. **头部**：
   - 节点名称（技术主题名或媒体名称）
   - 节点类型标签
   - 关闭按钮

2. **时间线标签页**：
   - 该节点的时间序列数据
   - 关键事件列表（带日期）

3. **关键词云标签页**：
   - 关键词云图
   - 关键词热度排行榜（前10）

4. **代表性报道标签页**：
   - 报道列表（标题 + 来源 + 日期）
   - 每条可展开查看摘要

---

### 1.8 热点检测与突发事件追踪

**位置：独立卡片区域（主内容下方）**

**布局：卡片列表**

**卡片内容：**
- 卡片标题："突发事件候选" / "新兴热点"
- 触发原因标签（红色高亮）
- 热度趋势折线图（mini版）
- 涉及技术/领域标签
- 操作按钮："加入监控" / "生成简报"

**卡片样式：**
- 宽度：1/3屏幕宽度
- 高度：200px
- 边框：左边框4px橙色（表示紧急程度）

---

### 1.9 动态监控与预警规则配置

**位置：点击左侧菜单"预警配置"时切换**

**布局：表单 + 列表**

**表单区域：**
- 监控对象类型：下拉框 [技术主题/科技实体/地域]
- 关键词输入框：支持多个关键词标签
- 指标选择：多选框 [提及数 / 正面情感比例 / 负面情感比例 / 媒体类型分布]
- 阈值设置：
  - 绝对阈值：输入框 + 单位
  - 相对阈值：百分比输入
- 时间窗：下拉框 [1小时/6小时/24小时/7天]
- 通知渠道：复选框 [站内消息 / 邮件 / 短信]
- 验证规则按钮：基于历史数据回测

**列表区域：**
- 已配置规则列表
- 每条规则显示：规则名称 / 监控对象 / 状态 / 创建时间
- 操作：编辑 / 删除 / 启用/禁用开关

---

### 1.10 报告生成

**位置：点击顶部"生成报告"按钮**

**布局：模态框**

**表单内容：**
- 报告模板：单选 [科技舆情简报 / 战略研判报告 / 舆情应对建议书]
- 时间范围：日期选择器
- 包含部分：复选框 [趋势分析 / 耦合分析 / 代表媒体 / 推荐行动]
- 导出格式：单选 [PDF / Word]
- 特殊要求：文本域（可空）

## 页面二：技术趋势分析

### 2.1 页面整体布局

---

### 2.2 顶部导航栏

**与页面一相同，模块导航选中项："技术趋势分析"**

---

### 2.3 左侧功能导航

**布局：二级菜单**

**一级菜单项：**
1. 技术体系分析
2. 论文分析
3. 专利分析
4. 科研项目分析
5. 社会网络分析
6. 技术趋势分析（当前激活）
7. 技术动态预警
8. 领域技术线路

**每个一级菜单的二级菜单示例（技术趋势分析）：**
- 聚类分析
- 共性网络分析
- 技术热点
- 交叉关联
- 合作分析
- 发展趋势

---

### 2.4 主内容区域 - 技术热点模块

**布局：搜索 + 筛选 + 列表 + 详情**

**搜索栏（顶部）：**
- 搜索框：输入关键词搜索技术主题
- 时间范围：下拉框 [2024/2023/2022/2021/2020/自定义]
- 技术领域：下拉框 [全部/人工智能/新能源/半导体/...]
- 标签页切换：[技术热点] [交叉关联] [合作分析] [发展趋势]

---

### 2.5 技术热点 - 列表区域

**布局：左右分栏（6:4）**

**左侧 - 热点技术排名列表：**
- 列表项结构：
  - 排名数字（1-5）
  - 技术名称（加粗）
  - 增长率标签（绿色上升/红色下降 + 百分比）
  - 所属领域标签
- 列表高度：滚动显示

**右侧 - 技术卡片组：**
- 卡片网格：2列 × 若干行
- 卡片内容：
  - 技术名称（大字标题）
  - 出现频次：数字 + 标签
  - 增长率：数字 + 趋势箭头
  - 论文数：数字
  - 专利数：数字
  - 关键词标签组
- 卡片操作按钮：[查看详情] [关联分析]

**卡片样式：**
- 圆角：16px
- 内边距：20px
- 悬停效果：轻微上浮 + 阴影加深

---

### 2.6 交叉关联模块

**布局：矩阵 + 网络图**

**左侧 - 交叉关联矩阵：**
- 表头：技术领域名称
- 单元格内容：技术组合名称 + 关联强度（色块深浅表示）+ 数值
- 颜色图例：强关联(深绿) / 中等关联(浅绿) / 弱关联(浅灰)

**右侧 - 关联强度分布网络图：**
- 节点：技术领域（圆形）
- 连线：表示关联，粗细表示强度
- 图例：强关联数量 / 中等关联数量 / 弱关联数量

**筛选控件（顶部）：**
- 下拉框：筛选关联强度 [全部 / 强关联 / 中等关联 / 弱关联]
- 导出按钮

---

### 2.7 合作分析模块

**布局：网络图 + 列表**

**左侧 - 合作网络可视化：**
- 节点：机构（圆形，大小表示合作数量）
- 连线：表示合作关系，粗细表示合作强度
- 颜色：按合作领域着色

**右侧 - 合作详情列表：**
- 列表项：
  - 机构名称（两个机构连在一起）
  - 合作论文数
  - 合作时长
  - 合作领域
  - 合作评分（星级或数字）
- 排序功能：按评分 / 按论文数 / 按时长
- 筛选功能：按合作领域

---

### 2.8 发展趋势模块

**布局：折线图 + 表格 + 统计**

**顶部 - 技术趋势折线图：**
- 图表类型：多折线图
- X轴：年份（2020-2024）
- Y轴：数量/指数
- 图例：各技术领域（不同颜色）
- 交互：点击图例显示/隐藏该曲线
- 高度：350px

**右侧 - 趋势数据统计：**
- 卡片列表（垂直排列）
- 每个卡片：技术领域名称 + 总增长率
- 顶部标签："趋势数据统计"

**底部 - 年度发展趋势对比表格：**
- 列：年份（2020-2024）+ 合计
- 行：各技术领域
- 单元格：数值 + 同比变化箭头
- 支持点击列标题排序

**筛选控件（图表上方）：**
- 时间范围选择器
- 技术领域多选
- 导出按钮

---

### 2.9 聚类分析子页面

**入口：左侧菜单"聚类分析"**

**布局：任务管理 + 结果展示**

**任务模块：**
- 顶部按钮：[创建聚类任务] [数据导入]
- 任务卡片列表：
  - 任务名称
  - 状态标签 [运行中/已完成/失败]
  - 创建时间
  - 操作按钮 [查看详情] [删除]

**结果模块：**
- 聚类结果卡片网格：
  - 聚类名称
  - 置信度进度条
  - 包含文档数
  - 代表性关键词标签（最多5个）
- 点击卡片 → 查看详情

**筛选机制模块：**
- 三个折叠面板：[初级筛选] [中间筛选] [最终筛选]
- 每个面板显示：筛选条件列表 + 权重值 + 结果预览
- 流程图区域：展示三级筛选的流转关系

---

### 2.10 共性网络分析子页面

**入口：左侧菜单"共性网络分析"**

**布局：任务 + 矩阵 + 规则**

**关系矩阵模块：**

**聚类规则模块：**
- 规则卡片列表：
  - 规则名称
  - 规则类型
  - 启用状态开关
  - 权重滑块
  - 词汇数量
- 顶部按钮：[添加规则] [批量配置]

## 页面三：技术动态预警

### 3.1 页面整体布局

**页面结构：**
- 顶部导航栏（与页面一、二相同）
- 左侧功能导航
- 主内容区域（4个主要模块）
- 底部状态栏

---

### 3.2 顶部导航栏

**模块导航选中项："技术动态预警"**

---

### 3.3 左侧功能导航

**布局：与页面二相同，选中"技术动态预警"**

**二级菜单：**
- 预警信息
- 动态跟踪
- 预警推送
- 趋势检测

---

### 3.4 预警信息模块

**布局：列表 + 详情**

**筛选控件（顶部）：**
- 搜索框：输入关键词
- 来源筛选：下拉框 [全部来源/政府发布/学术期刊/行业媒体/社交网络]
- 时间范围：日期选择器
- 技术领域：多选下拉框
- 置信度：滑块范围 [0-100]

**列表区域：**
- 卡片列表形式（垂直滚动）
- 每个卡片：
  - 预警标题（加粗）
  - 来源机构 + 发布时间
  - 置信度进度条 + 数值
  - 相关标签（技术领域）
  - 摘要文字（2行，溢出省略）
- 卡片左边框：按置信度着色（高=红/中=橙/低=蓝）

**详情面板（点击卡片展开）：**
- 详细内容区域
- 影响评估文字
- 相关技术/机构标签
- 操作按钮：[查看原文] [加入监控] [生成报告]

---

### 3.5 动态跟踪模块

**布局：左右双栏**

**左侧 - 技术动态栏：**
- 栏目标题："技术动态"
- 动态列表项：
  - 技术名称
  - 变化率（+45.2%，颜色表示正负）
  - 提及次数
  - 关键事件（链接形式）
  - 领域标签
- 按变化率排序

**右侧 - 专家动态栏：**
- 栏目标题："专家动态"
- 动态列表项：
  - 专家姓名 + 头像
  - 所属机构
  - 活动成果（论文/项目/奖项）
  - 时间
  - 影响描述
- 按时间排序（最新优先）

**公共功能：**
- 领域标签筛选（标签云形式）
- 时间排序切换 [最新优先/最旧优先]

---

### 3.6 预警推送模块

**布局：订阅管理**

**顶部操作区：**
- 搜索框：搜索订阅名称
- 按钮：[创建订阅]

**订阅列表：**
- 卡片列表形式
- 每个卡片：
  - 订阅名称（标题）
  - 关键词标签组
  - 创建时间
  - 最后预警时间
  - 状态开关（启用/禁用）
- 卡片操作：[编辑] [删除]

**创建订阅模态框：**
- 订阅名称输入框
- 关键词输入 + 标签添加
- 监控对象类型：[技术主题/专家/机构]
- 预警频率：[实时/每日汇总/每周汇总]
- 通知渠道：[站内消息/邮件/短信]

---

### 3.7 趋势检测模块

**布局：监测列表 + 预警建议**

**左侧 - 技术动态跟踪：**
- 与动态跟踪模块类似
- 额外显示：监测级别指示器

**右侧 - 预警建议：**
- 三个分类区域（垂直排列）：
  1. 高优先级关注（红色边框）
     - 技术名称 + 预警原因
     - 建议措施
     - 操作：[查看详情] [调整级别]
  2. 持续监测（橙色边框）
     - 技术名称 + 变化趋势
     - 建议措施
  3. 信息收集（蓝色边框）
     - 技术名称
     - 建议措施

**监测级别调整：**
- 下拉菜单：[高优先级 / 持续监测 / 信息收集 / 取消监控]
- 调整后自动移动到对应分类

---

### 3.8 预警通知中心

**位置：顶部导航栏通知图标**

**触发条件：点击通知图标**

**面板内容：**
- 通知列表（最新10条）
- 每条通知：
  - 预警图标（按级别着色）

## 交互规范

### 5.1 动效
- 卡片悬停：transform: translateY(-2px)，过渡0.2s
- 面板展开：slideIn 0.3s ease-out
- 按钮点击：scale(0.98) 0.1s

### 5.2 加载状态
- 骨架屏加载：灰白交替闪烁
- 数据加载中：中心旋转图标 + "加载中..."

### 5.3 空状态
- 图标 + 提示文字 + 操作按钮

### 5.4 错误状态
- 红色边框提示
- 错误信息文字
- 重试按钮

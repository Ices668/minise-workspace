## 架构特点

1. **四层分离**: 应用层 → 服务层 → 算法层 → 数据层，职责清晰
2. **三中台架构**: 业务中台+数据中台+AI中台，能力复用
3. **微服务**: 基于Spring Cloud的微服务架构，支持横向扩展
4. **异构存储**: 关系型+分析型+图+向量+全文检索，针对性选型

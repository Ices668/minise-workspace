主要关系:
• 科技实体 1:N 技术主题 (一个实体可属于多个技术主题)
• 科技实体 N:M 论文 (通过中间表关联)
• 科技实体 N:M 专利 (通过中间表关联)
• 技术主题 N:M 预警 (一个主题可触发多个预警)
• 趋势分析 1:N 报告 (一个分析可生成多份报告)
• 报告 1:N 附件 (报告可包含多个导出文件)

索引策略:
• 实体 name, type - 全文检索索引
• 论文/专利 title, abstract - 全文检索索引  
• 技术主题 name - B+树索引
• 趋势分析 created_at - 时间范围索引

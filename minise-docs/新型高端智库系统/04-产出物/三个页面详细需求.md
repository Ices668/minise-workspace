## 一、媒体聚焦页面

### 1.1 页面定位
将媒体舆情数据与科技实体清单、前沿技术分析结果与知识图谱深度联动，完成媒体信息与科技实体/技术主题的关联、耦合识别与动态监控，支持多层次（国际/国家/省市/城市/机构/学科）的可视化研判、预警与报告生成。

### 1.2 功能清单

| 序号 | 功能名称 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| 1 | 媒体聚焦总览面板 | 展示总监测媒体数、媒体关注热点数量、情感偏向指标（正/中/负占比）、媒体—技术耦合度总体评分 | P0 |
| 2 | 时间序列与趋势视图 | 展示媒体提及数、情感分布、学术产出（论文/专利/立项）、投融资事件、政策关键词出现频率的时间序列 | P0 |
| 3 | 媒体—技术耦合网络视图 | 以网络图展示媒体与科技实体/技术主题之间的关联强度，节点大小表示影响力或产出量 | P0 |
| 4 | 热点检测与突发事件追踪 | 基于短期增长、异常情感波动、来源集中度等规则识别突发舆情或新兴热点 | P0 |
| 5 | 耦合度分析与解读面板 | 展示媒体关注与技术发展指标之间的定量耦合分析：相关系数矩阵、滞后效应、Granger因果检验结果 | P1 |
| 6 | 动态监控与预警规则配置 | 支持主题/实体/地域自定义监控规则：阈值式、增速式、复合（情感+流量） | P0 |
| 7 | 报告与建议生成 | 支持按模板自动生成"科技舆情短报""战略研判报告""舆情应对建议书" | P1 |
| 8 | 智能推荐与溯源 | 基于知识图谱与语义相似度，推荐相关技术主题、潜在受影响机构、关键意见媒体/人物 | P1 |

### 1.3 交互说明

**媒体聚焦总览面板：**
- 点击"媒体关注热点"卡片 → 展开热点列表（按热度/增长率排序），支持将某热点加入监控清单
- 点击"耦合度评分" → 弹出耦合分析摘要，展示媒体关注与技术发展相关性的总体分布与显著耦合对

**时间序列与趋势视图：**
- 鼠标悬停显示某时间点详细指标
- 支持"多序列对比"：选择多条曲线并计算滞后相关系数
- 支持选择时间窗并导出该时间段数据与图表

**媒体—技术耦合网络视图：**
- 点击任一节点，显示其与媒体的时间线、关键词云与代表性报道
- 支持"局部放大"与"路径查找"（查找从某媒体到某技术主题的传播路径）
- 可按地域、时间范围、情感过滤网络

**热点检测与突发事件追踪：**
- 系统自动列出"突发事件候选"，并给出触发原因
- 用户可一键"加入突发监控"，或生成应对简报草稿

**动态监控与预警规则配置：**
- 配置窗口支持选择指标（提及数、正/负情感比例、媒体类型分布等）、阈值（绝对或相对）与时间窗
- 支持"验证规则"功能：基于历史数据回测规则有效性
- 告警可触发自动化动作：生成应急分析报告、通知指定人员、调用外部接口

**报告与建议生成：**
- 用户选择模板（简报/专业/决策层），选择时间范围与包含部分
- 支持导出 PDF/Word，或生成带权限的共享链接
- 历史报告可版本管理

### 1.4 数据来源
- 新闻媒体、社交媒体、研究报告等公开渠道信息
- 论文库（学术产出）
- 专利库（技术产出）
- 科研项目库（立项情况）
- 投融资数据库
- 政策数据库

## 二、技术趋势分析页面

### 2.1 页面定位
基于研究领域技术体系分析结果，利用深度挖掘算法，形成领域知识聚类清单，深度融合专家智慧，发挥专家的核心作用，广泛征求领域专家意见，逐步筛选出本领域关键的、迫切发展的初级技术清单、中间技术清单、最终技术清单，识别研究领域最活跃的研究前沿和发展趋势。

### 2.2 功能清单

| 序号 | 功能名称 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| 1 | 聚类分析 | 对科技数据进行聚类建模与多级筛选分析，支持创建聚类分析任务 | P0 |
| 2 | 共性网络分析 | 通过聚类分析规则与术语关系矩阵，对科技文献与专利数据进行语义网络构建与共性分析 | P0 |
| 3 | 技术趋势分析 | 通过论文、专利等多源科技信息的建模分析，实现对重点领域的技术热点与发展趋势的动态监测 | P0 |
| 4 | 技术热点模块 | 通过词云动态展示不同时期的高频技术方向 | P0 |
| 5 | 交叉关联模块 | 揭示领域间技术关联与融合趋势 | P0 |
| 6 | 合作关系模块 | 分析机构与学者间的科研协作结构 | P1 |
| 7 | 发展趋势模块 | 通过折线图与河流图等可视化手段，展示领域内技术演化脉络与未来走向 | P0 |

### 2.3 聚类分析子功能

**聚类任务模块：**
- 创建聚类分析任务（任务名称、数据来源、时间区间、关键词来源）
- 聚类任务卡片：任务名称、状态、描述、数据来源、时间范围
- 支持任务详情查看、聚类结果导出、清单重新计算

**聚类结果模块：**
- 聚类结果卡片：聚类名称、置信度、包含文档数量、代表性关键词
- 支持查看详情

**筛选机制模块：**
- 三级筛选机制：初级筛选、中间筛选、最终筛选
- 每个阶段展示筛选条件、权重值、产生的聚类结果
- 筛选流程图展示完整数据流转

**清单定制模块：**
- 创建清单：从聚类结果生成主题清单
- 清单项目：关键词/技术术语、相关度得分、出现频次
- 支持编辑、导出、重新计算

### 2.4 共性网络分析子功能

**分析任务模块：**
- 创建网络分析任务
- 分析任务卡片：任务名称、状态、数据来源、算法、创建日期
- 支持任务管理

**关系矩阵模块：**
- 术语关系表格：共现频次、语义相似度、上下文相关度、综合关系强度
- 支持按指标排序

**聚类规则模块：**
- 添加规则、批量配置
- 规则卡片：规则名称、类型、启用状态、权重、词汇数量

**语义网络模块：**
- 主题卡片：置信度、中心性、核心文献量、代表性关键词、高产作者

**高产作者模块：**
- 研究者信息卡片：姓名、所属单位、论文数、引用数、H指数、研究领域标签、代表性成果

### 2.5 技术趋势分析子功能

**技术热点模块：**
- 搜索栏、时间范围选择、技术领域选择
- 热点技术排名列表：技术名称、增长率、所属领域
- 详细信息卡片：出现频次、增长率、论文数、专利数
- 功能按钮：刷新数据、导出报告

**交叉关联模块：**
- 技术交叉关联矩阵：技术组合名称、关联强度等级、关联度数值、增长率、论文数
- 关联强度分布网络图
- 关联强度筛选：全部/强关联/中等关联/弱关联

**合作分析模块：**
- 合作网络可视化：机构间科研合作关系
- 合作详情列表：机构名称、合作论文数、合作时长、合作领域、合作评分
- 评分排序、领域筛选

**发展趋势模块：**
- 技术趋势折线图：2020-2024年各技术领域发展趋势
- 趋势数据统计列表：核心前沿技术领域及总增长率
- 年度发展趋势对比表格
- 时间范围选择器

### 2.6 算法设计要点

**聚类分析算法：**
- 数据预处理与基础向量化（稀疏表示+稠密表示）
- 关键词热度与趋势计算：带时间衰减的热度值与增长率
- IPC分类与部类分布分析
- 机构节点图与连边权重计算

**任务管理算法：**
- 任务数据模型与版本控制
- 状态机与幂等保证
- 优先级调度与资源控制
- 增量更新与合并策略

## 三、技术动态预警页面

### 3.1 页面定位
结合技术溯源与趋势分析，实现对潜在颠覆性技术的智能识别与动态预警。系统基于深度挖掘算法与信息抽取技术，对科技文献、报告、科研活动等内容进行智能跟踪，实时捕捉领域内的研究进展与专家动态。

### 3.2 功能清单

| 序号 | 功能名称 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| 1 | 预警信息模块 | 展示重大技术突破、创新进展及影响评估 | P0 |
| 2 | 动态跟踪模块 | 跟踪技术动态与专家动态 | P0 |
| 3 | 预警推送模块 | 专题定制与自动推送 | P0 |
| 4 | 趋势检测模块 | 技术热度变化监测与预警建议 | P0 |

### 3.3 预警信息模块

**功能：**
- 预警条目列表：标题、来源机构、发布时间、置信度及相关标签
- 筛选功能：按来源、时间范围、技术领域、置信度筛选
- 预警标签：标识预警所属技术领域
- 详情内容：原始研究论文、来源机构官网、相关技术分析链接

**交互：**
- 点击条目主体 → 展开完整预警信息
- 点击筛选控件 → 实时更新预警列表
- 点击标签 → 快速分类浏览

### 3.4 动态跟踪模块

**技术动态跟踪：**
- 技术名称、变化率（+45.2%）、提及次数（1256）、关键事件、领域标签
- 点击 → 跳转动态详情页

**专家动态跟踪：**
- 专家姓名、所属机构、活动成果、时间、影响描述、领域标签
- 点击 → 跳转专家详情页

**交互：**
- 关键事件链接 → 跳转事件详情或原文链接
- 领域标签 → 实时筛选
- 时间排序功能：按时间降序/升序

### 3.5 预警推送模块

**功能：**
- 创建订阅按钮
- 订阅状态开关：启用/禁用
- 订阅项主体：订阅名称、关键词、创建时间、最后预警时间
- 关键词标签：定义监控内容范围

**交互：**
- 点击创建订阅 → 填写订阅名称、设置关键词、选择预警频率
- 点击状态开关 → 切换订阅状态
- 点击关键词标签 → 查看相关动态列表

### 3.6 趋势检测模块

**功能：**
- 技术动态跟踪：技术领域、变化率、提及次数
- 预警建议分类：高优先级关注、持续监测、信息收集
- 监测级别调整控件

**交互：**
- 点击分类标签 → 展开该分类下的技术详情
- 点击技术名称 → 跳转预警详情页
- 点击监测级别调整 → 选择新监测级别

### 3.7 预警类型

| 预警类型 | 触发条件 | 告警级别 |
|----------|----------|----------|
| 热点突变 | 技术热度突然上升/下降超过阈值 | 告警 |
| 新技术出现 | 新技术主题首次被发现 | 通知 |
| 政策关联 | 技术主题与政策文件关联变化 | 告警 |
| 竞争动态 | 竞争对手技术布局变化 | 通知 |
| 风险预警 | 技术衰退/替代风险信号 | 严重告警 |

### 3.8 通知渠道
- 站内消息
- 邮件
- 短信
- 外部接口调用

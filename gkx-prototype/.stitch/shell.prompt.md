你在为一个大型政企数据平台设计 Web PC 端页面（1440px 宽）。

**DESIGN SYSTEM（必须严格遵守）：**
对齐参考截图风格：浅色政务/数据平台后台（但侧边菜单栏必须为白色）
- 顶部通栏 + 左侧白色侧栏 + 右侧浅灰背景内容区（白色卡片化）
- 主色青蓝/湖蓝，强调色亮青蓝
- 侧栏选中态：青蓝高亮块或左侧高亮条 + 轻微阴影，不要深色侧栏
- 圆角偏大，阴影很轻，留白大
- 筛选区胶囊输入框，表格在卡片内
- 全部中文文案

**页面：系统壳（顶栏+侧栏+内容区）**
目标：进入任一系统后的通用框架，展示导航层级：一级菜单=二级模块，二级菜单=三级模块。

**页面结构：**
1) 顶部通栏 Header：
- 左侧：Logo + 当前系统名（默认展示“战略咨询智能支持”）
- 中间：搜索框（placeholder：搜索…）
- 右侧：快捷入口（消息、帮助、通知）+ 用户头像下拉（个人中心/退出）

2) 左侧侧栏 Sidebar（深色渐变，带轻微纹理/光晕）：
- 分组标题：战略咨询智能支持
- 一级菜单（=二级模块）示例：
  - 研究需求分析
  - 技术体系分析
  - 技术趋势分析
  - 领域技术线路
  - 战略咨询报告
  - 战略咨询成果
- 当点击某个一级菜单后，显示二级菜单（=三级模块）列表；示例在“研究需求分析”下展开：
  - 科技实体清单（选中态）
  - 颠覆性与前沿技术分析
  - 媒体聚焦

**侧栏交互样式：**
- 选中二级菜单：亮青蓝高亮块/发光边缘，文字更亮
- hover：浅亮提示

3) 右侧内容区 Content：
- 面包屑：战略咨询智能支持 / 研究需求分析 / 科技实体清单
- 顶部标题：科技实体清单
- 筛选区卡片：搜索框（关键词模糊搜索）+ 下拉筛选（技术领域/地区/创新指数/标签）+ 重置/搜索按钮
- 工具栏：视图切换（网格/列表）、批量导入、导出
- 主体区域：列表视图（表格）+ 分页；表格列：名称、领域、创新指数、标签、更新时间、操作（查看/编辑/删除）

输出一个完整可展示的 HTML 页面。

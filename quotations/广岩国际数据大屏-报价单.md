**报价日期：** 2026年3月6日

## 一、项目概述

为广岩国际打造一套深色科技感数据可视化大屏（固定分辨率 1920×1080），包含 3D 交互地球、投资数据图表、成交项目滚动展示等模块。项目采用 Mock 静态数据演示效果，可用于展示、汇报、演示场景。

---

## 二、交付内容

| 序号 | 功能模块 | 说明 |
|------|----------|------|
| 1 | 3D 真实地球 | 可旋转，点击国家高亮并联动展示数据 |
| 2 | 侧边栏投资额展示 | 点击国家后显示该国投资额、基地数、项目数（Mock） |
| 3 | 投资方式占比图 | 环形/饼图，展示各类投资方式比例 |
| 4 | 每月交易趋势图 | 折线图/柱状图，展示月度成交趋势 |
| 5 | 最近成交项目滚动栏 | 自动滚动展示最新成交记录 |
| 6 | 大屏整体布局 | 深色科技感，1920×1080，Mock 数据填充 |
| 7 | 源码交付 | 完整前端源码（React + Vite + TypeScript） |

---

## 三、技术栈

- **框架：** React + Vite + TypeScript
- **3D 地球：** @react-three/fiber + three-globe
- **图表：** ECharts
- **动画：** Framer Motion / CSS 动画
- **布局：** Tailwind CSS（固定 1920×1080）

> ⚠️ 本报价为纯前端展示大屏，所有数据均为 Mock 静态数据，**不包含**后端开发、数据库及真实接口对接。
